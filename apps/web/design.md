# Public UI baseline

**Source**: reset public UI specification, pending the next approved design.

## Character

The public portfolio is intentionally plain and semantic. It has no branded theme,
decorative visual effects, special typography, or animation system.

## Build mandate

Keep public pages readable, responsive, accessible, and based on live API content.
Use the neutral rules in `src/app/globals.css` only until the next approved design
becomes the single visual source of truth.
