# Neon Toxic Archive

**Source**: Stitch project `9600445408490036302`, screen
`9316314ea1244ae7a061f0d590b8a72d`, Mark Angel Portfolio Neon Toxic Variant.

## Character

The public portfolio is a technical archive rendered as a high contrast cyber
interface. A near black grid canvas, toxic lime signals, violet diagnostics,
monospaced typography, scanline media, and square illuminated frames make the
page feel precise, active, and engineered rather than ornamental.

## Build mandate

Match the approved Stitch screen faithfully while binding every content surface to
the live portfolio API. Use the twelve column archive composition on desktop,
stack it deliberately on smaller screens, keep all effects behind readable content,
and preserve reduced motion, keyboard navigation, visible focus, live form states,
and semantic section landmarks. Token values live in `src/app/globals.css`.

## Composition

Use the fixed ARCHIVE.SYS navigation, split identity hero, selected works grid,
paired career and skills panels, paired education and credentials panels, system
capabilities, establish uplink form, and online archive footer from the Stitch
screen. Optional data sections remain conditional so the page never renders empty
frames.

## Component rules

Cards use square corners, one pixel lime or muted borders, restrained glow, and
violet only for diagnostic metadata. Media is grayscale with a scanline overlay and
a short glitch response on hover. Buttons read like system commands. Motion must
stop under reduced motion preferences.
